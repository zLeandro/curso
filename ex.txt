HTML: linguagem utilizada para estruturar os elementos da página, como parágrafos, links, títulos, tabelas, imagens e até vídeos.

CSS: linguagem utilizada para definir cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página

Javascript: linguagem de programação utilizada para deixar a página com mais movimento, podendo atualizar elementos dinamicamente e lidar melhor com dados enviados e recebidos na página.
